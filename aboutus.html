Ecommerce project

Developed with pure vanilla JS from the ground up. No database integration.

Description

This Ecommerce model is to showcase basic functionalities of an ecommerce site with the fake store api to display data. Each item can be added to the cart and therefore submit the order.

Functionalities

Fetched items using the Fetch Api.

Items are stored in localStorage. Therefore the quantity is displayed in the navbar.

Session storage is used to display messages when an item is added to the cart or when the order is submited.

There is a validation process when submiting the order, no fields must be empty.

If same item is added the quantity of the item updates.

Payment details are excluded.

Styling

CSS from the ground up, responsive design using media queries.
